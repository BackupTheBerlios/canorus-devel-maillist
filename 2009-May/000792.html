port) it would still be good to have a prototype test so an estimation
about the final performance can be made. If you or anybody else can show
some performance improvements in advance (I'm already content with

not within our own code (except you did some detailed performance
measuring with profiling tools.. but that might be more work than
implementing such a prototype).
