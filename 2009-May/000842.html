Hi everybody,

with r1083 importing midi is even more fun because time signatures are
now done right. Almost all my midi files are read in and displayed nicely :)
I'll soon improve exporting time signatures to the same level, which now
I understand better, so anacrusises will be handled properly too.

A little proposal (to Matevž): To keep the paper feeling of canorus,
which I really like, and keep canorus workable without any signatures or
barlines at all, and, nevertheless, keep a midi export/import cycle
close to identity, I would like to change the default tempo, which
probably also is the default playback tempo, to the midi default tempo.
The midi default tempo is quarter = 120.

The selection of pmidi as a parser was a good decision. The quantisation
should best be done only triggered by the user. The raw display in 128th
I like most, currently. It's the most unaltered sound. Probably we
should have some functions on the selected regions, like lowering
quantisation, merging voices, as Itay mentioned, shifting jazzy notes
horizontally, pressing notes to chords, and so on. Automatic
quantisation, or only to be selected prior to import, I don't think
that's what we want.


Best Regards,
Georg
