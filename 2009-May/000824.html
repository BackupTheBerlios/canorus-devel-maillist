You can use the progress bar by calling:
setStatus( positive status number )
setProgress( progress in percents )

Then reimplement readableStatus() method which returns the QString 
describing current status number. (eg. status number=1 would mean 
"Imporing voices", status number=2 would mean "Importing instruments" etc.).

All these is part of CAMidiImport class so you don't need to read the 
mainwin's progress bar code.
