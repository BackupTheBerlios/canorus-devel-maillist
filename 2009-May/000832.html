Hello everybody,

in r1076 midi export is improved in that tempo doesn't affect note
lengths anymore, as it always should have been. So we come closer to a
lossless midi export/import cycle. The next will be time signatures, now
still missing, then key signatures.

When we import a nice 3 minute medium combo piece, we see slowdowns
which should be tackled, for example when you want to change the tempo
of the first note. This should be discussed, maybe on irc.

Anyway, things are nicely improving,
