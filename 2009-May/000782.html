Hi Reinhard.

I think this feature should be triaged by a guitar user and programmer. 
I never actually read the tablature and chords, so I don't think I could 
be the right person to implement it.

The idea to take a look at kguitar is ok with me. If the original author 
doesn't want to port it to Qt4, this might be a lot of work for us. If 
you ask me, we should remake a native support for fret diagrams, chords 
and tablatures in Canorus. KGuitar should be a referenced project for 
how the things should be made. Directly borrowing the code from them 
might not be a good idea.
