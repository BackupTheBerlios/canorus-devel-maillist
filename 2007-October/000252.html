Hi Folks,

I'm a musician and a pretty good programmer.  I was looking for some
open source music composition/player code to which I could add some
features I've been wanting.

I'd like to add support for microtonal music.  In particular, I'd like
to see not only quarter-tones, but sixth-tones, or any other tones one
might imagine. This could also be used to let people use different
temperaments for playing music.

It would also be nice to have better control of tempi, such as long
accelerandos or decelerandos that are not just linear.  For example, an
accelerando that starts gradually and speeds up much faster at the end,
or a change in tempo that speeds up and slows down in waves over the
course of the whole piece of music.  Such an implementation might also
allow for different styles of playing, such as the swing style one
person was looking for.

Anyway, as you are still developing this, I thought it would be useful
to see if I could add to your project. I'd be happy to help in whatever
way I can if the project would allow a useful way of composing with
microtones.

Dan Grubbs
