Hi Dan,

I'm happy to read this. Microtonal music should not be too hard to
implement. First thing you want to do is change accidentals() in CANote
from int to float.  Second is probably the GUI for this (which btw. we
currently only offer +/- keys for accs, no icons etc. yet - this should
go into playable toolbar). And the third thing, playing (setting midi
tune value?) and import/export and printing using LilyPond. Lily already
supports quarter-tone
http://lilypond.org/doc/v2.11/Documentation/user/lilypond-big-page#Micro-tones
so we should keep its compatibility at least for quarter-tones.

Expression mark context (for tempo, dynamics etc.) is on my TODO list
for version 0.5 and should be implemented someday in November :). The
non-linear tempo sounds very tempting. Maybe we should write envelopes
for dynamic and tempo.


Regards.
-Matevž
