I've attached the error report that Apple's Font Book program gave when
I tried to install the font (renamed to "Emmentaler-14.old.ttf"). You
can probably ignore the one about there being another Emmentaler
already installed; there is, it's the one that works :)

Here's what I did in fontforge to the .ttf supplied with the canorus
source (this also works with Emmentaler-14.otf from Lily):

* Under "Element/Font Info" in the "Names" tab, change both the
"Fontname" and "Name For Humans" to "Emmentaler".

* Under "File/Generate Fonts" choose "True Type" from the left popup
menu and then click "Options":

  - UNcheck "Apple" (and "Opentype" if it is checked) - ironic huh?
  - CHECK "Save Comments", "Save Colors", "TeX Table", "Output Glyph
Map", "Output OFM & CFG"

I don't think all of those last check boxes are needed, but I didn't
test all the combinations to see which ones specific ones worked.

> Do you know of any linux tools which convert or save icons in .icns
> format? It might be useful for the future as well.

No, all the tools I could find either convert FROM icns, or use Cocoa
APIs for the conversion which means they only work on OS X. There is
one program that came up, img2icns, that could possibly be ported to
GNUStep, but I imagine that would be a fair bit of work. It is REALLY
easy to use Icon Composer to convert transparent PNGs to .icns though
(takes 10 seconds drag & drop).
