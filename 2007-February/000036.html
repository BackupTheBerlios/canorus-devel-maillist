> Hey,
> I have the following problem. I work in eclipse. If I build the canorus
> project and run it, the program uses gtk1 and not qt. Is that right? Or,
> is the layout from qt equal with the layout from gtk1.

You can't run Canorus with gtk1 instead of qt, since it is written in
qt. Qt can use gtk as a backend, so if it runs you probably did it
alright.

> An other problem
> is, that the clef (and notes) are chinese symbols.

In order to see the music symbols you must install the fonts found in
the src/fonts/ directory.
