Hi,

Just added support for pure Lily syntax import/export, R355. Work is
based on our old CanorusML syntax, but it's now pure LilyPond. To use it:
1) write some notes
2) View->View voice in LilyPond
3) source view opens up with lilypond syntax

Parser is quite complex but I think it's worth it. See
import/lilypondimport and export/lilypondexport. It's Thread ready,
collects errors/warnings through the process etc.


Please test!
