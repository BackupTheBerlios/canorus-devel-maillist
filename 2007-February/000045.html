>> be found in /usr/share/fonts/truetype on Linux and C:\Windows\Fonts on

Emmentaler-14.ttf is located in src/fonts directory. You can check it
out from svn
http://svn.berlios.de/wsvn/canorus/trunk/src/fonts/
or from the 0.1.0 release - it's there.

You should copy this .ttf file to /usr/share/fonts/truetype and fonts
should automagically work.
