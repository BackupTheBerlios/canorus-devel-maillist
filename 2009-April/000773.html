Hi Pavel,

Thanks for your generous offer to translate Canorus into czech!
We are using Qt4 as development base and use their translation
system as well.
Some language editors support these tranlations files AFAIK but
if you're already familiar the linguist this might be the best
option to use (except if you already have a language dictionary
for musical terms -- I don't know if it's possible to convert
such a dictionary into qt linguist's own format).

The language files are in the trunk/src/lang folder (".ts").
The README.Translators file contains help instructions.
You can send us the translations files either as patch or
(preffered option) directly on the user or development mailing
list ( http://developer.berlios.de/mail/?group_id=6144 ).

Thanks again for offering help,

Reinhard Katzmann
