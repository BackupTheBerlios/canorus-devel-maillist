The default document location on Windows should be
C:/Documents and Settings/David Bolton/My Documents

(Where "David Bolton" is the name of the user profile.)
In the current SVN the default document location is
C:/Documents and Settings/David Bolton/

The root profile folder in Windows contains system files not intended 
for normal viewing. The root profile folder would probably confuse 
casual computer users.

Also the left-hand column is missing default locations present in native 
open/save dialogs such as "My Recent Documents," "Desktop," "My 
Documents," "My Computer," and "My Network Places." See the screenshots 
attached for a comparison of a native dialog (from Microsoft Word 2003) 
and the current Canorus dialog. Although I like some of the features of 
the new open/save dialogs in the SVN, the open/save dialogs in the 
stable branch had a more native look and feel.
