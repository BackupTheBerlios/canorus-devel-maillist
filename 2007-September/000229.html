Hello to all,

I updated our nightly builds server so that it copies the latest
template.ts to the root directory of nightly builds.
How to translate Canorus to a new language:
1) download template.ts from our nightly build site
(http://canorus.jekovec.net) or build the latest SVN version and find it
in src/lang directory.
2) translate the .ts file using Qt linguist or other compatible tool
3) rename .ts file to yourlanguage_YOURCOUNTRY.ts
4) send the translation to devel mailing list or me personally

Current languages being worked on:
- English: default
- Slovenian: me
- German: Georg
- Hebrew: Itay
- Dutch: raboof
- Swedish: Magnus

Thanks for your efforts!


Regards.
