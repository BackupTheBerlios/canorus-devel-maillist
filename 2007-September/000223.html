R578 brings the following clefs:
- G clefs: treble, french
- F clefs: bass, subbass, varbaritone
- C clefs: soprano, mezzosoprano, alto, tenor, baritone

Note that the implementation *broke clefs in saved documents*. If you
have any documents saved, open it in Canorus, delete existing clefs,
place new ones and resave.

language.
