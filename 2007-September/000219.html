Canorus development team is happy to announce the new release of Canorus
- a free cross-platform music score editor, version 0.4beta. This
release is meant for testers to track down all the bugs and for
developers to fix them :)

Download beta at:
http://developer.berlios.de/project/showfiles.php?group_id=6144

Changes:
- Settings dialog
- Customizable colors for Score editor
- Element advanced properties dialog
- Copy/Paste
- Undo/Redo
- Splash screen
- MIDI instrument per voice support
- Key signature major/minor support and function marking integration
- Scripting engine is now always synchronized with the current API
- Python engine now uses objective Swig interface
- New CanorusML support in scripting library for directly opening/saving
Canorus files
- New playback support in scripting library
- Qt4.3 compatibility
- lots of smaller additions and even more fixes

Testers please use bugtracker:
https://developer.berlios.de/bugs/?group_id=6144

Translators, please compile the source application, edit template.ts
with Qt Linguist and save your translation as language_COUNTRY.ts. After
recompilation, translations should be immediately available. Please send
translations to me personally or this mailing list for the 0.4 release.

Needed translations:
- German (Reinhard, Georg?)
- Hebrew (Itay?)
- Sweden (Magnus?)
