Hi Christian,

Canorus currently saves/loads into its own format (called CanorusML) and
LilyPond format. CanorusML is XML based file format and can be found in
src/core/canorusml.cpp/h. LilyPond is a non-xml file format (which you
might be more interested in) and can be found in
src/import/lilypondimport.cpp/h and src/export/lilypondexport.cpp/h.

You might start be examining how LilyPond export and import of Canorus
music is done.


Regards.
