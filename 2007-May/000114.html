Hi guys,

I'm glad to inform you that the last month has been very productive
indeed. I finally managed to implement the lyrics support I had in mind
for so long and it works great. LilyPond export and lyrics source view
works too. I'm pretty sure that we have the most comprehensive lyrics
there have ever been in music score editors ;) The one feature still
missing but not currently much interested is the custom associated voice
for each syllable. Some stuff is out there for this (like
CASyllable::associatedVoice()), but the functionality is not implemented.

Please test :)
