Hi Georg.

I'm currently in designing how to implement the auto bar stuff. I'd do
something like when you insert a new note, it determines the current
time signature and the last barline and if the timestart of the new note
and the last barline is too big, it adds a barline in front of the new note.

I'll also refactor a few methods inside CAVoice. There will only be 4
methods for all the query stuff. All others will be its derivatives.
- CAVoice::next(CAMusElement*)
- CAVoice::nextByType(CAMusElementType, CAMusElement*)
- CAVoice::previous(CAMusElement*)
- CAVoice::previousByType(CAMusElementType, CAMusElement*)

And if the CAMusElement* is null, it returns:
- next(null) the first element in the voice
- previous(null) the last element in the voice

What do you think?
