Hi guys.

We're currently having a discussion of how to implement the resources
inside the document with Itay.

I played around a bit and in the end, we might expect something like this:
1) Insert->Picture...
2) User selects the picture (supported all image formats by Qt4) and
inserts it
3) Picture appears in the document and on the resources list
4) User prints the document
5) The image is converted into a EPS format (using QPrinter)
6) The image is inserted into the lilypond document using
\markup{\epsfile #X #10 #"./image.eps"}
7) PDF is generated (or printed). See the attachment for the result.

Reinhard, you might be interested in the QImage->EPS conversion. See the
attached script.
