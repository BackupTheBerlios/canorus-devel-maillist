recovery file, the new instance will recover the file. I think that it
shouldn't be possible to have more than one instance running - new instances
will create a new document in a new mainwindow of the old instance and then
exit.

above. You can't see the note, and there are no scrollbars.

work. But then if you commit changes, and then window->unsplit all, the only
score viewport is unsplitted (did I do it?). It doesn't always happen
though. Also it looks like interaction with the source view text box doesn't
make it the current view port, and window->close current view closes the
score view port instead.
