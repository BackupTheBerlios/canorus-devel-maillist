Midi out is ok. The problem is that the redraw of the viewport is not as
fast as it could probably be. So the whole CPU goes to refreshing the
viewport and software synth has lower priority then and playback stutters.

You can solve this by:
1) increase software synth buffer size, decrease sample rate (I did this
for fluidsynth and works ok)
2) using an external Midi device for playback (soundcard capable of
hardware midi playback or a synth). Same can be done with timidity.
