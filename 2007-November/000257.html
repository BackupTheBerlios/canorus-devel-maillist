I'm still here.  I'm new to linux, so getting set up took a while, but
I've finally got the whole thing compiled and running.

I hadn't thought of envelopes for dynamics, but that seems as important
as variable tempos.

If someone puts me on as a developer, I could get started doing
something.  My user name at berlios is: 'dgrubbs'.  I'd be happy to work
on whatever you think would be most useful at the moment, though I do
have a suggestion...

The midi output, at least on my computer, is terrible.  It misses the
first few notes, and barely sounds the rest of them; and this is with

I don't know how well it works on other systems, but it seems important
that the midi output works correctly.  It would be hard to test some of
the other stuff without it.  I don't know if others have this problem,
or if someone else is working on this.
