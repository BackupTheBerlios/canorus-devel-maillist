Just a list of bugs I came across the other day (Build R621_Win32):

* Horizontal scrolling does not automatically accommodate for notes 
entered beyond one screen view. (Scrolling is fine if you save, close, 
and reopen file).

* In the Save As dialog, the file name input box is erased every time 
you switch folders (if folder is selected from list on right-hand side).

This last one is probably more of a feature request:

* If a time signature is present it would be nice for the bar lines to 
be added automatically. The bar lines are added automatically in the 
final lilypond rendering even if they are missing in the original 
Canorus file.
