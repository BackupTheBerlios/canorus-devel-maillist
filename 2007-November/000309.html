Thanks for the feedback. For now I have only a small tip regarding KBabel. You
should use the dictionary (KBabelDict) for getting a similar feature like the
phrase book. There is a automatic search function you can switch on/off but for
larger dictionaries it tends to get slow.
Also when you start translation you might want to try the automatic translation
feature if you already have some kind of database. As told in my last mail I'd
load a noteedit po file for the language you create and add it's terms to the
database you can later use these translations for canorus too.
