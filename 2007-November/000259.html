Hi everyone, I've been lurking for a while keeping up to date with 
progress as I haven't had time to contribute a useful amount of time to 
the project.

However, I've been working with microtonal music (specifically 19-tone 
ET) for the past couple of years, and have some code lying around which 
might help save a few hours for anyone adding microtonal support.

The tuning class (in C++) converts between pitch, frequency & MIDI. It 
is suitable for a system which retunes MIDI notes to frequencies (as in 
MIDI tuning standard or fluidsynth) rather than systems which aim to use 
the pitch bend (which aren't really much use anyway).

Tunings can be read from a text file, I also have an XML implementation 
of the tuning definition which I could dig out (writing has yet to be 
supported). It works for non-equal tempered tunings, and I suspect it 
would work for tunings which are not based on the octave, but I haven't 
had the need to use these yet, so I'm not sure.

If this is of interest/use I have put the relevant classes and an 
example tunings file in an archive at the location below:
http://userweb.elec.gla.ac.uk/d/dougiemc/tuningclass.tgz

If you have any questions feel free to ask, please let me know if you 
decide to use it, or if you improve the code. I'll be away travelling 
soon, so my replies might be slightly delayed.


Cheers,
