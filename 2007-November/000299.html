Hi,

When researching abc2xml / xml2abc (this stuff exists, it just was hard to
find) I came across Sonata. Interesting because it uses Qt4 and has some score
editing features in it. Apparently it uses the tools above to convert abc
scores to xml and via verse. These tools I thought might serve as inbetween
solution for some of our import/export stuff (or as a base for implementing
it). Unfortunately the sources stuff except the musicxml lib (being GPL)
contain neither copyright nor any license hint. I assume Public Domain until
I find furhter hints.

Here is the main page of sonata:

http://ciant114.cesnet.cz/sonata/?ACT=About

I plan to upload the original abc2xml stuff as archive on the Canorus download
page as I didn't find any other place where to get it except from the sonata
source (there it is a part of a complete package).
