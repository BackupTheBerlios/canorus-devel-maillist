Hi,

I'd like to know if you find it useful to have shortcuts for the
recent file list. I'm personally unsure about it.

One the one hand, you cannot know the number beforehand as the
range might change very often. Also it's not easy to get the
list as it resides in a sub menu.

On the other hand I often find it clumsy to open recent files
using the mouse. Even with 'alt f' -> 'r' you have to navigate
with cursor to open one of the files.

Another drawback is that this way I wouldn't be able to store
these action settings (not even manually).
