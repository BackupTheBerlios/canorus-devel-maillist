Basically, you always copy instances of CAMusElement class or their
subclasses.

It contains a pointer to the context which the element is part of so you
can know which context a music element belongs to when copying. It's
just a matter of implementation - which stuff to get cloned and which not.

See reference:
http://canorus.jekovec.net/doc-stable/classCAMusElement.html
