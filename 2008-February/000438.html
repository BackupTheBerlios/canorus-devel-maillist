Hi guys.

I added CADiatonicPitch, CAPlayableLength and CAInterval classes which
describe note pitch (note name+accidentals), playable length
(length+number of dots) and interval (perfect fourth, major second etc.).

I'm currently implementing CAInterval functionalities, so we can sum
pitch+interval=new pitch and implement transpose function then. The main
reason why I did this from the first place is the work on Harmonia - I
need lots of note and interval manipulation and it's the best to have it
implemented in C++ Canorus library and not in Python.

I also renamed CAFunctionMarking and CAFunctionMarkingContext to
CAFunctionMark and CAFunctionMarkContext.

I've rewritten the half of notes and rests stuff so I probably broke
something or a lot ;) Please test.
