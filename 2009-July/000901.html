CMAKE_BUILD_TYPE
  Specifies the build type for make based generators.

  This specifies what build type will be built in this  tree.  Possible
  values are  empty, Debug, Release, RelWithDebInfo and MinSizeRel. This
  variable is only supported for make based generators. If this variable
  is supported, then CMake will also provide initial values for the vari‐
  ables with the  name   CMAKE_C_FLAGS_[Debug|Release|RelWithDebInfo|Min‐
  SizeRel].   For   example,   if   CMAKE_BUILD_TYPE   is   Debug,   then
  CMAKE_C_FLAGS_DEBUG will be added to the CMAKE_C_FLAGS.
