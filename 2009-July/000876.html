Hi Georg.

I fixed the windows build problem today and the compiler error lied in 
pmidi/except.h. It uses sigjmp_buf, which is defined only on POSIX 
systems in setjmp.h. On Windows, it's jmp_buf only.

The same problem was found in Wine:
http://www.winehq.org/pipermail/wine-devel/2003-November/022093.html

And the solution in Wine is now:
http://source.winehq.org/git/wine.git/?a=blob;f=include/wine/exception.h;h=7f013490a8086b4093cf6493b7783ac4d192871c;hb=refs/heads/master
