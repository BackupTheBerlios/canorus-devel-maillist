Hi,

I have no idea what the problem is here regarding the installation
and the font. I though a simple make install works for all *nix systems.
Any ideas ?

Also I vote for removing "executing of src/canorus" and replace it by
a proper short "make install" and run "canorus". Running src/canorus
for Compilation looks strange. What about

Compilation
-----------
...
$ make

Installation
------------
...
$ make install

Running
-------
...
$ canorus

And add "if the program is not found make sure that the directory
containing the canorus executable is in your $PATH (or %PATH%).
Also note that Windows has a maximum path length limitation, so
if possible we should avoid to change the path on that platform at all.
