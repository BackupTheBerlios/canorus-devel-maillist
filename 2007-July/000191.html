Matevž wrote:
> Canorus currently uses 4 distinct modes: Select mode (default), Edit
> mode, Insert mode and Playback mode. I think those modes are so
> elemental that can't and shouldn't be unified. When changing modes, the
> pretty obvious border of the score viewport changes so no "mode errors"
> should be done so easily. Mouse cursor should also be changed in the future.
>
>   
I think we might have a different understanding of modes. My working 
definition is where the same behavior (by the user) produces different 
results depending on the mode. Under this definition the "Insert mode" 
you mention is technically three separate modes: "Insert new staff 
context," "Insert new lyric context," "Insert new function marking 
context." An then there is

I agree that Canorus does a good job of making modes visible but I think 
"mode errors" may be more common than you think (perhaps in a different 
way than you where thinking).

The mixing of slurs and ties in Finale is one example of mode error.

Another example of a mode error in Canorus is trying to enter a note on 
the bass clef while the treble clef is selected. The result is 
unexpected, but because I know about this particular modalism I can 
easily recover.

For a newbie it may be difficult to even get started as evidenced by 
Martin's question on the user list a month ago. Martin complaining that 
he couldn't work out how to enter a simple melody. The user expectation 
was that note entry would be clearly visible on the interface. The quick 
fix was to have the first staff selected by default upon opening. My 
preference would be for Playable (and other context tools) to always be 
visible and clicking on the screen inserts the note on the nearest staff.

Another type of mode error is differences between programs. For whatever 
reason Finale expects users to click on the staff instead of the actual 
lyric to select a lyric syllable. Although it is more logical to click 
on the actual syllable to edit a lyric (as is the behavior in Canorus), 
humans are habit forming I must remember I am in "Canorus mode" rather 
than "Finale mode" to overcome this habit.
