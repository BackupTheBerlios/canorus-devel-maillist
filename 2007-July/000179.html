Question: is it better to use the discussion lists, private email, or 
the bug tracking system for reporting minor bugs? For what its' worth, I 
have a few more suggestions below.

- Use file extension specific to Canorus (such as ".can" or ".canorus") 
instead of the generic ".xml". This way it could automatically associate 
with Canorus when opening.

- Add tenor clef (this is occasionally used by trombones, cellos, etc. 
to avoid ledger lines). Also consider adding octave clefs particularly 
treble clef with an 8 below it (used for tenor line in 
Soprano-Alto-Tenor-Bass settings)

- In other score editors I've tried (i.e. Finale, Sibelius) hyphen 
automatically jumps to the next syllable for lyrics. Unless there is a 
good rational for avoiding this behavior, Canorus should do the same.
