> Because of existent beta software limitations, at the moment canorus is only delivered with an export option. But anyway, you can use the import plugin in the following way.
>   
Import document menu option is now enabled in R526. Sorry for this... I
implemented the code behind import a long time ago, but forgot to enable it.
> 1. Start canorus
> 2. choose Menu-> file -> export (or STRG+E)
> 3. choose in file format (cap import filter)
> 4. choose a capella 2000 file (e.g. one of the example files)
> 5. commit file overwrite message
>  :-)
>
> In this development state, only Notes will be displayed.
> Sometimes the keys will not be visible, but if you save the imported file as canorus xml file and reload this file, the keys are visible.
>
