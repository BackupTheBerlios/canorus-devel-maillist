I'm doing more research with poppler and printing. I meanwhile
have set Rendering options concerning anti aliasing expizit to
on and found the result is now much better (it seems to me though

interpreted as text and I didn't have TextAliasing set to on).

uses poppler too, but so far no printing is done via poppler,

does not run on win32 (as it does not paint the file).

I haven't yet looked into kprint stuff and evince though
being based on GTK/Gnome might deliver some rendering hints
or informations concerning printing itself.
