Hi,

I've changed a lot to the file handling code. I'm forced
to use another file (copy of the temp file) which will be
overwritten when a new print or a new preview is started
(print.pdf, preview.pdf) as I have at least for preview no
possibility to get any information when the started preview
process is finished.

I have changed the resolution from PrintResolution to
ScreenResolution as PrintResolution is deprecated. I was
unable to use HighResolution without waiting either an
endless time or getting memory exhaustion without swap
activated. According to the docs 1200dpi are set as a
fixed resolution. I'm really disappointed about that fact
that the real printer resolution is ignored and that the
complete image is rendered to RAM first before it is printed.

I hope that the class offers more possibilities to achieve
my goals as up to now without qt4 printing poppler example
I can only guess about many things.
