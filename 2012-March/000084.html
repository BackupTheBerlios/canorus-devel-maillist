> and 37096 is open.  Don't know how to proceed.

This one is easy at least :) OS X does not like the style of TrueType
fonts included with the Canorus distribution; download a tweaked
version of Emmentaler here: http://merphant.net/canorus . Install the
font to /Library/Fonts, not $HOME/Library/Fonts. Pretty soon I'll
upload a modified version of all of the fonts for Canorus, probably to
the macosx/ dir in svn along with some build system stuff. I'll also
update the installation notes on the wiki.
