I forgot, you should also put <refresh/> tag inside <action> stanza,
this will rebuild the GUI when the script is done (notes will be drawn
out of the data part of the document you create in the script).

>         <action>
>                 <name>pluginTestInit</name>
>                 <import-filter>My file format (*.mus)</import-filter>
>                 <lang>python</lang>
>                 <filename>myimportscript.py</filename>
>                 <args>import-filename</args>
>                 <args>document</args>
>   
                            <refresh/> <!-- add this! -->
>         </action>
>
