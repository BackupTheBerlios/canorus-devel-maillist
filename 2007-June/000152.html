Hi Matevž,
Great job!

Here are a few issues I noticed already:
1/ after adding a tie I can't add any new notes no matter what I do until I
select another context.

2/ Insert a new context -> Insert an element to the context -> Undo twice ->
Insert a new context -> crash.

3/ Insert a bunch of notes -> Playback everything -> While playing back,
click undo or redo twice -> crash.

4/ Add voice -> Undo -> crash

5/ Window splitting through the window menu doesn't work anymore.

6/ A common shortcut for Redo (esp. in linux) is ctrl-shift-z, which is IMO
much more convenient the ctrl+y. I think it should also be supported.

7/ Removal of an additional Sheet can be undone, while removal of the only
one cannot.

8/ CanorusML view doesn't show up.

9/ Insert a note -> Change stem direction -> Undo twice -> Insert a note ->
crash

10/ Insert a note with an accidental -> Insert another note -> Undo -> Watch
the accidental turn gray -> Click on the gray accidental (as if inserting a
note there) -> crash.

That's it for now.

Sorry I don't have the time to help more than just reporting the bugs...
