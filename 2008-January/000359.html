Hello everybody,

midi export is basically working now, and done with a lot of code reuse,
which I liked! What's still missing is proper export of time signatures,
so that importing programs don't have to guess too much about the
quantization of the music, I guess, and the export of lyrics. Even more
could be missing.

Embedded texts, like header text, should be translated, and other texts
should be in locale, but still managed in these 8 bit strings, which I
don't understand fully yet. Probably there is a simple trick with
QTextStream. Currently stream.codec("Latin-1") avoids UTF encodings when
outputting binary QByteArrays to QTextStream, but that might not work in
non european countries, hmm.
