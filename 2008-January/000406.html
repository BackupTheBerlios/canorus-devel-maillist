Hi,

a small bug you can see like this:

Start canorus, insert these two quarters in the first staff, which is 
C-major: as and g. Hit the edit button (e), go with the left cursor 
button to the as.
1: Only the flat sign is highlighted, not the note itself. Hitting the 
cursor up will not move the note up.

2: If you hit cursor down, first the note will be highlighted too, the 
the note will move down as expected.

3: Now cursor up will move the note up too.

That's it, could be worse :)
