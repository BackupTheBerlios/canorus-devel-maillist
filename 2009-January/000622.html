Hi,

In german these shortcuts are actually translated (ctrl -> strg), but
this is already done by the generic Qt translation (for automatic
translation actually).

You can override shortcuts this way but we intend to make a keyboard
and midi (for midi input) dialog where the user can change these default
settings anyway. Of course even then doubling of shortcuts in translation
has to be avoided (I've seen many applications with double shortcuts,
so it's not naturally done).
