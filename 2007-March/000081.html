We'll release the final user's guide when Canorus 1.0 is out. Me and
Reinhard were talking about this and we'll probably write the user's
guide in plain LateX format in lyx (a very nice latex frontend). No
user's guide has been written or started yet though. The documentation
will be parsed then and a PDF created (and available by using
Help->User's guide).

the user's guide .ly document.
Whereas for developer's guide, I'll probably extend one or more wiki
page for developers.
