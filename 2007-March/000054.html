Hi again, here's another patch for your review: this adds basic
Lilypond file export support, building on the voice export functions.
For now it only exports the first sheet of a document, but it should be
easy to add the rest. Also I tweaked some numbers for the pitch C in
the voice import and export functions; it was exporting an octave too
low and importing an octave too high.

I haven't tested it extensively but it exports the example-major.xml
and example-minor.xml files from Harmonia with no errors.
