Hi Patrick.

Canorus currently uses default Windows location for storing the settings.
See
http://doc.trolltech.com/4.3/qsettings.html

chapter "Platform-Specific Notes".

That is
%APPDATA%\Canorus\config.ini

where %APPDATA% is usually C:\Documents and settings\<user>\Application
data.

You could simply set the variable APPDATA to something else and Canorus
would store its settings there:
$ set APPDATA=C:\something else
$ canorus.exe

Is this usable?


Regards.
