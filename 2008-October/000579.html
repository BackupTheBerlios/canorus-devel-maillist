Hi guys.

I've been rethinking the plans and decided to cancel the 0.6 release.
I'd like to see the version 0.7 coming out around New year. I agree
with Georg about the main editing and stability stuff. I'd also like
to have printing support there and ready for end users when the
release is out. And finally there are some other features I currently
need for my composition (will talk about them later) so I think we
should focus more on development and testing for the next few months
than releasing ASAP.

Current users still have nightly builds for their favourite platform
if they want to try the latest features.

What I'd also like to see is the documentation project to be started
for 0.7. I remember one posting the Latex examples of our users guide.
Was there any additional work done on this?


Regards.
-Matevz
