Hi Reinhard.

Yes, I agree. I just don't know how you would achieve this easily.

For example, if I have a
CAMusElement *elt
I want to know if it's a note, I do:
if ( elt->musElementType()==CAMusElement::Note ) ...
and from now on, I can use
static_cast<CANote*>(elt).

The only way I can think of replacing this is to add a new casting
method, for example:
CANote *CAMusElement::note() { return dynamic_cast<CANote*>(this); }

So if I had
CAMusElement *elt
and I would like to know if it's a note, I now do:
if (elt->note()) { elt->note()->setPitch... }

Any other ideas?

btw. Python types should work fine.
