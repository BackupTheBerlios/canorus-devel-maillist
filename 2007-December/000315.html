Hi guys.

I made the following plan for Canorus 0.5:
- 2nd January: feature freeze - translators should start translating
- 14th January: release candidate, extensive bug chasing, translations
submitted
- 20th/21th January: 0.5 release.

Major things done:
- staff consistency, robustness, stability
- threaded import/export
- import/exported opened to scripting engine

Major things to be done:
- dynamics, tempo changes
- printing (Reinhard?)
- viewports reorganization (Itay?)

Other stuff, if we have time:
- staccato, marcato, portato etc. on notes
- fermata
- arpeggio
- octaviation
- pedal markings

New translations:
- French
- Danish
- Norwegian
- German
- Traditional Chinese


Regards.
-Matevž
